Yet Another (nasty) Chip8 Assembler
-----------------------------------

Just a small assembler for Chip8/SuperChip8 processors.

## supported instructions: ##

- ADD
- LD
- CLS
- AND
- OR
- XOR
- SUB
- SUBN
- JMP/JP
- CALL
- SCD
- SCR
- SCL
- LOW
- HIGH
- RET
- SE
- SNE
- SHR
- RSB
- SHL
- MVI
- JMI
- RND
- DRW
- SKP
- SKNP
- SYS
- EXIT
- DB
- DW
